# Roberto Reif Website Style Guide

## Color Palette

The website uses a minimal color scheme with:

- **Background**: White/light neutral tones
- **Text**: Dark gray/black for primary content
- **Accent**: LinkedIn blue (#0077B5) for social links
- **Logo**: Multi-colored (brand identity)

## Typography

Based on the Squarespace template configuration:

- **Primary Font Family**: System fonts (unspecified in inline styles)
- **Font Sizes**: Responsive scaling implemented via "TextShrink" utility
- **Font Weights**: Standard weights (not explicitly defined in provided data)
- **Logo Text**: Uses dynamic shrinking for mobile optimization

## Spacing & Layout

- **Outer Padding**: 100px (desktop)
- **Page Padding**: 80px
- **Top Padding**: 80px
- **Responsive Breakpoint**: 640px width for mobile adjustments
- **Grid Structure**: Centered, single-column design with asymmetric padding

## Structural Elements

- Header navigation with collapsible menu
- Dual logo placement (upper and lower sections)
- Horizontal dividers between content sections
- Image containers with no explicit border styling

## Interactive Styling

- **Social Icon Wrapper**: Uses `box-shadow: 0 0 0 2px inset` for border effects
- No explicit shadow or gradient styles in core CSS

## Notable Features

- Google Fonts loading framework (TypeKit integration available)
- reCAPTCHA styling support
- Mobile-optimized text scaling
- Minimal decorative styling, emphasizing content clarity

## Design Philosophy

The website follows a clean, minimal aesthetic that prioritizes:
- Content readability
- Responsive mobile-first design
- Subtle interactions
- Professional presentation
